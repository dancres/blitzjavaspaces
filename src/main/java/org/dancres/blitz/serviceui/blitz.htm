Blitz Dashboard ServiceUI

The Blitz Dashboard ServiceUI is available via the Factories/JFrame menu option.

Features

  • Memory monitor
  • Entry instance count, with break down for each entry type in the JavaSpace
  • Entry instance count history graph
  • Read,write & take counts, with a break down for each entry in the JavaSpace
  • Read,write & take counts pie chart & history views
  • Active Transaction count + history chart
  • Number of blocking takes + number of blocking reads

Running multiple Blitz ServiceUIs

Please note that if you have multiple Blitz JavaSpace instances running on the same host, with a shared codebase, then you must first close the dashboard window before connecting to a different Blitz instance.
